# 💡 추가 팁: 드론 스트림처럼 사용하기

다운로드한 영상을 Python에서 드론 스트림처럼 처리할 수 있어:

```python
import cv2

cap = cv2.VideoCapture("drone_video.mp4")

while True:
    ret, frame = cap.read()
    if not ret:
        break
    cv2.imshow("Drone Video", frame)
    if cv2.waitKey(1) == 27:
        break
```

➡ 실제 드론 없이도 **영상 분석 개발** 가능.
