
A1. set과 sorted 활용
Python
nums = [1, 2, 2, 3, 4, 4, 5, 6, 6] result = sorted(list(set(nums)), reverse=True) print(result) # [6, 5, 4, 3, 2, 1]

A2. dictionary get 메서드 활용
Python
text = "apple banana apple cherry banana apple" words = text.split() word_count = {} for word in words: word_count[word] = word_count.get(word, 0) + 1 print(word_count)

A3. List Comprehension
Python
import math result = [math.sqrt(x) for x in range(1, 21) if x % 3 == 0 and x % 2 == 0] print(result) # [math.sqrt(6), math.sqrt(12), math.sqrt(18)]

A4. Dictionary Filtering
Python
scores = {"철수": 90, "영희": 75, "민수": 85, "지수": 60} top_students = [name for name, score in scores.items() if score >= 80] print(top_students) # ['철수', '민수']

A5. 가변 인자와 복수 반환
Python
def get_range(*args): return max(args), min(args) high, low = get_range(10, 20, 5, 40, 33) print(f"최대: {high}, 최소: {low}")

A6. Set Operations
Python
list1 = [1, 2, 3, 4, 5] list2 = [4, 5, 6, 7, 8] s1, s2 = set(list1), set(list2) print(f"교집합: {s1 & s2}, 합집합: {s1 | s2}")

A7. Lambda Sort
Python
people = [("Alice", 25), ("Bob", 20), ("Charlie", 30)] people.sort(key=lambda x: x[1]) print(people)

A8. Split and Reverse
Python
s = "Python is powerful" result = " ".join(s.split()[::-1]) print(result)

A9. Dictionary Comprehension
Python
data = {"A": 1, "B": 2, "C": 3} inverted = {v: k for k, v in data.items()} print(inverted) # {1: 'A', 2: 'B', 3: 'C'}

A10. enumerate 활용
Python
medals = ['Gold', 'Silver', 'Bronze'] for i, medal in enumerate(medals, 1): print(f"{i}등: {medal}")
